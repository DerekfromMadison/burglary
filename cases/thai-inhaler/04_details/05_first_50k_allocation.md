## Rationale Highlights

**Amazon 28%** is upper-end for unvalidated wedge. If Phase 0 conditional, trim to $9–10k; reallocate to retention (SMS/email).

**Erewhon 12%** is PR ROI, not revenue. No smoothie collab year one.

**Creator 18%.** Pailin Chongchitnant (@hotthaikitchen) as anchor if opted in — gifted + supplier-story + advisor seat, not paid.

**Phase-1 paid social ($5k)** is distinct from Phase 0's $3k. Phase 0 establishes; Phase 1 scales.

## Absent — Intentional

Documented sub-$250k sinks:

- **Meta prospecting.** Burns without validated angle.
- **Podcasts.** Mid-six-figures for measurable lift.
- **Packaging redesign.** v1 lasts 18 months.
- **Sephora.** Slotting + co-op + PO > $50k.
- **PR retainer.** $5–15k/mo marginal.
- **TikTok Shop affiliate at scale.** 25% comm + 6% ref + ship = $7.91 contribution on $24 2-pack.
- **Chain retail.** Saje owns Ulta. Vicks owns CVS. Apothékary owns Shiseido-backed Ulta+Sprouts+WF.
