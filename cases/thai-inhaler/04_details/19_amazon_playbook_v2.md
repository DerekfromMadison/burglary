## 2. The USPTO IP Accelerator path (NEW, the unblock)

USPTO's [Patents/Trademarks for Humanity](https://www.uspto.gov/initiatives/expanding-innovation/patents-and-trademarks-humanity) and the parallel **Trademark Modernization Act expedited examination** (TEAS Plus + Special Petition for Make Special) routes have a $100 fee and target 12-week first action versus the 8–10 month standard track. The IP Accelerator path is the only structurally viable way to get a registered (or active-pending) trademark in time for Amazon Brand Registry approval inside the 90-day plan.

**Timeline (IP Accelerator path)**:
- Day 1: File TEAS Plus + Special Petition. $100 special-petition fee + $250/class TEAS Plus = $350 USPTO total.
- Day 1: File Brand Registry application using IP-Pending track (Amazon accepts an active TM application as long as it's progressing through USPTO).
- Week 5–8: USPTO first action lands (12-week target).
- Week 6–9: Brand Registry approval lands behind first-action.
- Week 9: A+ Content + Vine + Sponsored Brands all unlock.
- Week 9–12: Vine claims live (30 units at $200), reviews begin posting day 8–45 of claim window.

**Without IP Accelerator (standard track)**:
- Day 1: File TEAS Plus standard. $250/class.
- Month 8–10: USPTO first action.
- Month 9–11: Brand Registry approval.
- **The entire 90-day window passes without Brand Registry. A+ Content, Vine, and Sponsored Brands do not exist for the launch cohort. Amazon delivers 0–8 customers max via naked Sponsored Products on a base listing that competes nakedly on $14 vs $4 Poy-Sian and loses.**

The $100 IP Accelerator fee is the highest-ROI line item in the entire 90-day budget. v6 deliverable allocates it as part of the $1,500 Amazon line.
